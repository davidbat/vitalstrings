Description
Discover the updated Watson Sparse Matrix Package, a collection of algorithms for efficiently solving large systems of linear equations whose coefficient matrices are sparse. This software can be used as a serial package, or in a shared-memory multiprocessor environment, or as a scalable parallel solver in a message-passing environment.
