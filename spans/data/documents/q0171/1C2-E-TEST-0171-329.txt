    What is an IP Address?
    IP Address is an Internet Protocol address (IP address) which is a numerical label assigned to each device (e.g., computer, notebook, tablet, printer) which is connected to a computer network that uses the Internet Protocol for communication.
   An IP address have two principal functions: host or network interface identification and location addressing. Its role has been characterized as follows: A name indicates what we seek. An address indicates where it is. A route indicates how to get there.
